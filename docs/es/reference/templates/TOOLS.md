<div id="toolsmd-local-notes">
  # TOOLS.md - Notas locales
</div>

Las habilidades definen *cómo* funcionan las herramientas. Este archivo es para *tus* detalles específicos: las cosas que son únicas de tu configuración.

<div id="what-goes-here">
  ## Qué poner aquí
</div>

Cosas como:

- Nombres y ubicaciones de las cámaras
- Servidores y alias SSH  
- Voces preferidas para TTS
- Nombres de altavoces/salas
- Alias de dispositivos
- Cualquier cosa específica de tu entorno

<div id="examples">
  ## Ejemplos
</div>

```markdown
### Cameras
- living-room → Main area, 180° wide angle
- front-door → Entrance, motion-triggered

### SSH
- home-server → 192.168.1.100, user: admin

### TTS
- Preferred voice: "Nova" (warm, slightly British)
- Default speaker: Kitchen HomePod
```


<div id="why-separate">
  ## ¿Por qué separar?
</div>

Las habilidades se comparten. Tu configuración es tuya. Mantenerlas separadas significa que puedes actualizar las habilidades sin perder tus notas y compartir habilidades sin exponer tu infraestructura.

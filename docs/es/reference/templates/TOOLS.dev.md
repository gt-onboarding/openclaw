<div id="toolsmd-user-tool-notes-editable">
  # TOOLS.md - Notas del usuario sobre herramientas (editable)
</div>

Este archivo es para *tus* notas sobre herramientas externas y convenciones.
No define qué herramientas existen; OpenClaw proporciona herramientas integradas de forma interna.

<div id="examples">
  ## Ejemplos
</div>

<div id="imsg">
  ### imsg
</div>

- Envía un iMessage/SMS: indica a quién/qué y confirma antes de enviar.
- Prefiere mensajes cortos; evita incluir secretos.

<div id="sag">
  ### sag
</div>

- Texto a voz: especifica la voz, el hablante o la sala de destino y si debe transmitir en streaming.

Añade todo lo demás que quieras que el asistente sepa sobre tu toolchain local.

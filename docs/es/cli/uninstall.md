<div id="openclaw-uninstall">
  # `openclaw uninstall`
</div>

Desinstala el servicio Gateway y los datos locales (la CLI se mantiene).

```bash
openclaw uninstall
openclaw uninstall --all --yes
openclaw uninstall --dry-run
```

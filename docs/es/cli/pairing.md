<div id="openclaw-pairing">
  # `openclaw pairing`
</div>

Aprobar o inspeccionar solicitudes de emparejamiento de mensajes directos (DM) (para canales que admiten emparejamiento).

Relacionado:

* Flujo de emparejamiento: [Emparejamiento](/es/start/pairing)

<div id="commands">
  ## Comandos
</div>

```bash
openclaw pairing list whatsapp
openclaw pairing approve whatsapp <code> --notify
```

<div id="openclaw-reset">
  # `openclaw reset`
</div>

Restablece la configuración y el estado locales (mantiene instalada la CLI).

```bash
openclaw reset
openclaw reset --dry-run
openclaw reset --scope config+creds+sessions --yes --non-interactive
```

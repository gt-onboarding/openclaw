<div id="openclaw-dns">
  # `openclaw dns`
</div>

Utilidades DNS para descubrimiento en áreas extensas (Tailscale + CoreDNS). Actualmente centrado en macOS + Homebrew CoreDNS.

Relacionado:

* Descubrimiento del Gateway: [Descubrimiento](/es/gateway/discovery)
* Configuración de descubrimiento en áreas extensas: [Configuración](/es/gateway/configuration)

<div id="setup">
  ## Configuración
</div>

```bash
openclaw dns setup
openclaw dns setup --apply
```

<div id="openclaw-tui">
  # `openclaw tui`
</div>

Abre la UI de terminal conectada al Gateway.

Relacionado:

* Guía de la TUI: [TUI](/es/tui)

<div id="examples">
  ## Ejemplos
</div>

```bash
openclaw tui
openclaw tui --url ws://127.0.0.1:18789 --token <token>
openclaw tui --session main --deliver
```

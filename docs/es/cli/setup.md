<div id="openclaw-setup">
  # `openclaw setup`
</div>

Inicializa `~/.openclaw/openclaw.json` y el espacio de trabajo del agente.

Relacionado:

* Primeros pasos: [Getting started](/es/start/getting-started)
* Asistente de incorporación: [Onboarding](/es/start/onboarding)

<div id="examples">
  ## Ejemplos
</div>

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

Para ejecutar el asistente con `setup`:

```bash
openclaw setup --wizard
```

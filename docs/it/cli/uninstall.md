<div id="openclaw-uninstall">
  # `openclaw uninstall`
</div>

Disinstalla il servizio Gateway e i dati locali (la CLI resta disponibile).

```bash
openclaw uninstall
openclaw uninstall --all --yes
openclaw uninstall --dry-run
```

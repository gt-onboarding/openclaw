<div id="openclaw-setup">
  # `openclaw setup`
</div>

Inizializza `~/.openclaw/openclaw.json` e lo spazio di lavoro dell&#39;agente.

Contenuti correlati:

* Per iniziare: [Per iniziare](/it/start/getting-started)
* Procedura guidata: [Onboarding](/it/start/onboarding)

<div id="examples">
  ## Esempi
</div>

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

Per eseguire la procedura guidata tramite il comando setup:

```bash
openclaw setup --wizard
```

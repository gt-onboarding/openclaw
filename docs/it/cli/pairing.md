<div id="openclaw-pairing">
  # `openclaw pairing`
</div>

Approva o esamina le richieste di abbinamento per i messaggi diretti (DM) (per i canali che supportano l&#39;abbinamento).

Correlato:

* Flusso di abbinamento: [Pairing](/it/start/pairing)

<div id="commands">
  ## Comandi
</div>

```bash
openclaw pairing list whatsapp
openclaw pairing approve whatsapp <code> --notify
```

<div id="toolsmd-user-tool-notes-editable">
  # TOOLS.md - Note utente sugli strumenti (modificabile)
</div>

Questo file è destinato alle *tue* note sugli strumenti esterni e sulle convenzioni.
Non stabilisce quali strumenti sono disponibili; OpenClaw fornisce strumenti integrati.

<div id="examples">
  ## Esempi
</div>

<div id="imsg">
  ### imsg
</div>

- Invia un iMessage/SMS: descrivi chi/cosa, conferma prima dell’invio.
- Preferisci messaggi brevi; evita di inviare dati sensibili.

<div id="sag">
  ### sag
</div>

- Sintesi vocale: specifica la voce, l’altoparlante o la stanza di destinazione e se usare lo streaming.

Aggiungi qualsiasi altra cosa che vuoi che l’assistente sappia sulla tua toolchain locale.

<div id="toolsmd-user-tool-notes-editable">
  # TOOLS.md - 用户工具笔记（可编辑）
</div>

此文件用于记录*你自己*关于外部工具和约定的笔记。
它并不用于定义有哪些工具；OpenClaw 在内部已经提供了内置工具。

<div id="examples">
  ## 示例
</div>

<div id="imsg">
  ### imsg
</div>

- 发送 iMessage/SMS：说明收件人和内容，在发送前确认。
- 尽量使用简短消息；避免发送敏感/机密信息。

<div id="sag">
  ### sag
</div>

- 文本转语音：指定声音、目标说话人/房间，以及是否进行流式输出。

添加任何你希望助理了解的本地工具链相关信息。

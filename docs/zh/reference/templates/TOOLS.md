<div id="toolsmd-local-notes">
  # TOOLS.md - 本地备注
</div>

技能定义了工具*如何*工作。这个文件用于记录*你的*特定内容——也就是你这个环境中特有的配置和说明。

<div id="what-goes-here">
  ## 这里放什么
</div>

比如：

- 摄像头名称和位置
- SSH 主机与别名  
- TTS 首选语音
- 扬声器/房间名称
- 设备昵称
- 任何与特定环境相关的内容

<div id="examples">
  ## 示例
</div>

```markdown
### Cameras
- living-room → Main area, 180° wide angle
- front-door → Entrance, motion-triggered

### SSH
- home-server → 192.168.1.100, user: admin

### TTS
- Preferred voice: "Nova" (warm, slightly British)
- Default speaker: Kitchen HomePod
```


<div id="why-separate">
  ## 为什么要分离？
</div>

技能是可以共享的，而你的设置只属于你自己。把它们分开，你就可以在不丢失笔记的情况下更新技能，也可以在不泄露你的基础设施的前提下共享技能。

---

把任何能帮你完成工作的东西都加进来。这是你的速查表。

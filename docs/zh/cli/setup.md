<div id="openclaw-setup">
  # `openclaw setup`
</div>

初始化 `~/.openclaw/openclaw.json` 和智能体工作区。

相关内容：

* 快速开始：[快速开始](/zh/start/getting-started)
* 向导：[上手引导](/zh/start/onboarding)

<div id="examples">
  ## 示例
</div>

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

要通过 setup 命令运行向导：

```bash
openclaw setup --wizard
```

<div id="openclaw-pairing">
  # `openclaw pairing`
</div>

Approuver ou inspecter les demandes d’appairage de messages directs (DM) (pour les canaux qui prennent en charge l’appairage).

Voir aussi :

* Flux d’appairage : [Pairing](/fr/start/pairing)

<div id="commands">
  ## Commandes
</div>

```bash
openclaw pairing list whatsapp
openclaw pairing approve whatsapp <code> --notify
```

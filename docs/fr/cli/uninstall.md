<div id="openclaw-uninstall">
  # `openclaw uninstall`
</div>

Désinstaller le service Gateway ainsi que les données locales (la CLI est conservée).

```bash
openclaw uninstall
openclaw uninstall --all --yes
openclaw uninstall --dry-run
```

<div id="openclaw-setup">
  # `openclaw setup`
</div>

Initialise `~/.openclaw/openclaw.json` et l&#39;espace de travail de l&#39;agent.

Voir aussi :

* Premiers pas : [Premiers pas](/fr/start/getting-started)
* Assistant de configuration : [Onboarding](/fr/start/onboarding)

<div id="examples">
  ## Exemples
</div>

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

Pour lancer l’assistant via `setup` :

```bash
openclaw setup --wizard
```

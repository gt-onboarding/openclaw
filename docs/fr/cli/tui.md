<div id="openclaw-tui">
  # `openclaw tui`
</div>

Ouvre l&#39;UI en mode terminal connectée au Gateway.

Liens associés :

* Guide TUI : [TUI](/fr/tui)

<div id="examples">
  ## Exemples
</div>

```bash
openclaw tui
openclaw tui --url ws://127.0.0.1:18789 --token <token>
openclaw tui --session main --deliver
```

<div id="openclaw-uninstall">
  # `openclaw uninstall`
</div>

Gateway サービスおよびローカルデータをアンインストールします（CLI はそのまま残ります）。

```bash
openclaw uninstall
openclaw uninstall --all --yes
openclaw uninstall --dry-run
```

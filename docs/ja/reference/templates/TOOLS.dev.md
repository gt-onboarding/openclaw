<div id="toolsmd-user-tool-notes-editable">
  # TOOLS.md - ユーザー用ツールメモ（編集可）
</div>

このファイルは、外部ツールや運用上の慣習について、*あなた自身*がメモを書いておくためのものです。
ここで利用可能なツールの有無を定義するわけではありません。利用可能なツール自体は、OpenClaw が内部にビルトインとして用意しています。

<div id="examples">
  ## 使用例
</div>

<div id="imsg">
  ### imsg
</div>

- iMessage/SMS を送信: 誰に / 何を送るかを明示し、送信前に必ず確認すること。
- 短いメッセージを心がけ、秘密情報は送信しないこと。

<div id="sag">
  ### sag
</div>

- テキスト読み上げ（text-to-speech）：使用する音声、対象の話者／部屋、およびストリーミングするかどうかを指定します。

ローカルのツールチェーンについて、アシスタントに共有しておきたい情報があれば、自由に追記してください。

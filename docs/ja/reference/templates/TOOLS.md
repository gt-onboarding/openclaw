<div id="toolsmd-local-notes">
  # TOOLS.md - ローカルメモ
</div>

スキルは、ツールが*どのように*動作するかを定義します。このファイルには、*あなた*の環境に固有の内容——あなたのセットアップに特有の情報を書き留めてください。

<div id="what-goes-here">
  ## ここに記載する内容
</div>

次のようなもの:

- カメラ名と設置場所
- SSH ホストとエイリアス  
- TTS に使う優先する音声
- スピーカー／部屋の名前
- デバイスのニックネーム
- その環境に固有のあらゆる情報

<div id="examples">
  ## 使用例
</div>

```markdown
### Cameras
- living-room → Main area, 180° wide angle
- front-door → Entrance, motion-triggered

### SSH
- home-server → 192.168.1.100, user: admin

### TTS
- Preferred voice: "Nova" (warm, slightly British)
- Default speaker: Kitchen HomePod
```


<div id="why-separate">
  ## なぜ分けるのか？
</div>

スキルは共有されます。セットアップはあなたのものです。両者を分けておけば、ノートを失うことなくスキルを更新でき、インフラ情報を漏らすことなくスキルを共有できます。

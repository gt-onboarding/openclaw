<div id="agentsmd-openclaw-workspace">
  # AGENTS.md - OpenClaw ワークスペース
</div>

このディレクトリはアシスタントの作業ディレクトリです。

<div id="first-run-one-time">
  ## 初回セットアップ（1回のみ）
</div>

- `BOOTSTRAP.md` が存在する場合は、その手順に従い、完了したら削除してください。
- エージェントのアイデンティティは `IDENTITY.md` にあります。
- ユーザープロファイルは `USER.md` にあります。

<div id="backup-tip-recommended">
  ## バックアップのヒント（推奨）
</div>

このワークスペースをエージェントの「メモリ」として扱う場合は、identity やノートがバックアップされるように、Git リポジトリ（可能であれば非公開）にしておいてください。

```bash
git init
git add AGENTS.md
git commit -m "Add agent workspace"
```


<div id="safety-defaults">
  ## 安全デフォルト
</div>

- シークレットや機密データを持ち出さないこと。
- 明示的に要求されない限り、破壊的なコマンドを実行しないこと。
- チャットでは簡潔にし、長い出力はこのワークスペース内のファイルに書き出すこと。

<div id="daily-memory-recommended">
  ## 日次メモリ（推奨）
</div>

- 短い日次ログを memory/YYYY-MM-DD.md に記録します（必要に応じて memory/ ディレクトリを作成）。
- セッション開始時に、存在する場合は当日分と前日分を読み込みます。
- 永続的な事実、嗜好、決定事項を記録し、秘密情報は記載しないでください。

<div id="heartbeats-optional">
  ## ハートビート（オプション）
</div>

- HEARTBEAT.md にはハートビート実行用の簡潔なチェックリストを記載して構いませんが、内容はできるだけ小さく保ってください。

<div id="customize">
  ## カスタマイズ
</div>

- ここにお好みのスタイルやルール、「メモリ」などを追加してください。

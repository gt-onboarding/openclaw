<div id="openclaw-setup">
  # `openclaw setup`
</div>

Initialisiert `~/.openclaw/openclaw.json` und den Agent-Arbeitsbereich.

Zugehörig:

* Erste Schritte: [Erste Schritte](/de/start/getting-started)
* Einrichtungsassistent: [Onboarding](/de/start/onboarding)

<div id="examples">
  ## Beispiele
</div>

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

Um den Einrichtungsassistenten mit setup auszuführen:

```bash
openclaw setup --wizard
```

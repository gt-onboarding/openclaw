<div id="openclaw-uninstall">
  # `openclaw uninstall`
</div>

Entfernt den Gateway-Dienst und die lokalen Daten (CLI bleibt erhalten).

```bash
openclaw uninstall
openclaw uninstall --all --yes
openclaw uninstall --dry-run
```

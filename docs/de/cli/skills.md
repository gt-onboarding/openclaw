<div id="openclaw-skills">
  # `openclaw skills`
</div>

Fähigkeiten inspizieren (mitgelieferte, aus dem Arbeitsbereich sowie verwaltete Overrides) und prüfen, welche einsatzbereit sind und bei welchen noch Anforderungen fehlen.

Verwandte Themen:

* Fähigkeiten-System: [Skills](/de/tools/skills)
* Fähigkeiten-Konfiguration: [Skills config](/de/tools/skills-config)
* ClawHub-Installationen: [ClawHub](/de/tools/clawhub)

<div id="commands">
  ## Befehle
</div>

```bash
openclaw skills list
openclaw skills list --eligible
openclaw skills info <name>
openclaw skills check
```
